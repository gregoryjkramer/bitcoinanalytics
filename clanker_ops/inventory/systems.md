# Tenacious Data Sovereign Intelligence Platform v1 — Systems Inventory

_Last updated: 2026-07-02_

## Mission

Build a functional, revenue-generating clanker system for Tenacious Data that turns owned data, governed context, Bitcoin analytics, BI/Fabric expertise, and AI execution into practical products and consulting leverage.

The near-term objective is not an agent swarm. The near-term objective is one hardcore operator that reliably converts rough strategic intent into documented architecture, GitHub issues, implementation checklists, Codex prompts, and shipped artifacts.

## Primary Operating Principle

One useful clanker first. More clankers only after the first operator proves it can ship repeatable work.

## Forbidden Zones

The clanker must not touch the following without explicit human approval and a separate high-friction procedure:

- Bitcoin private keys
- Wallet seed phrases
- Lightning seed material
- Fedimint guardian secrets
- Passwords
- API keys
- Production DNS changes
- Financial accounts
- Tax filings
- Legal filings
- Irreversible deploys that affect money, identity, or customer data
